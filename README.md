Buscador de Recetas - Christian Eduardo Mendoza Marin

##Descripción

Aplicación web para buscar recetas de cocina internacional utilizando la API de TheMealDB. Permite explorar recetas por categorías y guardar favoritos.

##Características
- Búsqueda de recetas por categoría
- Visualización de recetas con imágenes
- Sistema de favoritos con LocalStorage

##Tecnologías Utilizadas
- HTML5
- CSS3 (con gradientes y animaciones)
- JavaScript (Vanilla JS)
- TheMealDB API
- LocalStorage para favoritos
